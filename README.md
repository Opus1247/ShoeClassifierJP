# ShoeClassifier
My name is JP and for my project, I am making a shoe classifier with the Jetson Orin Nano. The ShoeClassifier AI uses the re-trained resNet model to classify popular sneakers.
# Algorithm
Based on the [Popular Sneakers Classification dataset]([url](https://www.kaggle.com/datasets/nikolasgegenava/sneakers-classification)) it classifies images of sneakers.
# Requirements
1. Jetson Orin Nano
2. Models
3. Dataset
